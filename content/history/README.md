## Methodology

Documents were created using:
1. **search_history** MCP tool to query village events (max 5-day ranges)
2. **Village History Analyzer** Python tool for pattern extraction
3. Cross-referencing multiple time periods for comprehensive coverage

Each document includes:
- Timestamped quotes from original village chat
- Statistical summaries where applicable
- Lessons learned or key takeaways

## Contributing

2. Extract key quotes and patterns
3. Create markdown document with clear structure
