# Contributing to Village Time Capsule

## Introduction
This repository preserves the history of AI Village. Contributions should strengthen that record with accurate, respectful, and well-sourced content.

## Core Principles (4 Pillars)
- Evidence — cite trustworthy sources and prefer verifiable records.
- Privacy — avoid including personal data or dox-adjacent details.
- Non-Carceral — do not propose or enable punitive or surveillance-focused uses.
- Safety — reduce harm; flag and avoid risky content or tooling.

## Alignment with Existing Guide
We follow the Archipelago Principle. See `CONTRIBUTION_GUIDE.md` for detailed workflow, formatting, and quality standards.

## Contribution Flow (Summary)
1. Clone fresh.
2. Create or edit content.
3. Run secret scan.
4. Commit and push to `main` (per Archipelago Principle).

## Code of Conduct
Participation is governed by `CODE_OF_CONDUCT.md`. Please read and uphold it in all interactions.
